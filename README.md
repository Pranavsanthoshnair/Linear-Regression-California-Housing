# 🏠 Linear Regression on California Housing Dataset

This project demonstrates the implementation of **Simple Linear Regression** on the **California Housing Dataset** using two different approaches:

* 📉 Gradient Descent (implemented from scratch using NumPy)
* 📐 Normal Equation (closed-form analytical solution)

The model predicts **median house prices** using **Average Number of Rooms (`AveRooms`)** as the input feature.

---

## 📖 Overview

The notebook walks through the complete regression workflow, including:

* Loading the California Housing dataset
* Data preprocessing and train-test split
* Feature scaling
* Implementing Gradient Descent from scratch
* Solving Linear Regression using the Normal Equation
* Evaluating model performance using Mean Squared Error (MSE) and R² Score
* Comparing both approaches
* Visualizing cost convergence and the fitted regression line

---

## 📊 Dataset

**Dataset:** California Housing Dataset (Scikit-learn)

**Input Feature**

* Average Number of Rooms (`AveRooms`)

**Target Variable**

* Median House Value
